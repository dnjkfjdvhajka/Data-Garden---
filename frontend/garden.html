Data Insights

Design Philosophy

This project combines traditional Chinese aesthetics with modern data visualization to create an immersive sensory experience. Each flower represents a specific time period, connecting ancient Chinese timekeeping with contemporary data analysis.

Data Story

The data collected represents sensory experiences throughout a 24-hour period, showing how different environmental factors and human perceptions change over time. By exploring the visualizations, you can discover patterns in how our senses respond to different times of day.

Technical Approach

Using Chart.js for data visualization and a responsive design approach, this project transforms abstract data into an intuitive, interactive experience. The backend API provides real-time data from a MySQL database, ensuring accurate and up-to-date information.
